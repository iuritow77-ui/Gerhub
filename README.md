# Gerhub - Gerenciamento de Jigs

Este projeto é uma página HTML que exibe as informações dos **jigs cadastrados** em uma planilha.  
O objetivo é facilitar a visualização e o gerenciamento dos dados de forma simples e acessível via navegador.

## 🚀 Funcionalidades
- Exibição dos jigs cadastrados em formato de tabela.
- Leitura dos dados diretamente de uma planilha.
- Interface simples e intuitiva para consulta.

## 📂 Estrutura do projeto
- `visualizar_jig.html` → Página principal que mostra os dados.
- `css/` → Estilos da página (se houver).
- `js/` → Scripts para manipulação dos dados (se houver).
- `planilha/` → Arquivo com os dados dos jigs (se estiver incluído).
